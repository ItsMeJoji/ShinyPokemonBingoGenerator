How to Use...

  • When you load or refresh the page, you should get 15 randomized Pokemon with randomized Generation in each box of the Bingo Card.
  • Clicking on the Pokemon image will randomize that specific box.
  • Filling out the Fields on top will allow you to set the Space, Pokemon, and Generation of your choice. Click Set to set the center box.
  • Clicking on the Export as PNG button will generate a PNG image of the Bingo Card.
  • Clicking on the Go Back... button will take you back to the main page.
  • Remember to check if the Pokemon is obtainable in the game you are playing!

    • TLDR; refresh to randomize, click to randomize, set any space as you like using Space Number (1-25), Pokedex Number (1-1025), and Generateion from dropdown, remember to check if Pokemon is obtainable!
