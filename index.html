How to Use...

  • When you load or refresh the page, you should get 15 randomized Pokemon with randomized Generation in each box of the Bingo Card.
  • Clicking on the Pokemon image will randomize that specific box.
  • Filling out the Fields on top will allow you to set the Pokemon and Generation of your choice. Click Set to set the center box.
  • Clicking on the Export as PNG button will generate a PNG image of the Bingo Card.
  • Clicking on the Make a Custom Bingo Card button will take you to a page where you can create your own Custom Bingo Card.
  • Remember to check if the Pokemon is obtainable in the game you are playing!

    • TLDR; refresh to randomize, click to randomize, set the center as you like using Pokedex Number (1-1025) and Generation from dropdown, remember to check if Pokemon is obtainable!
